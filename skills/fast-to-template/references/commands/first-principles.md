# 第一性原理解释

从最底层的基本概念开始，逐层向上构建理解。

## 输出结构

1. **基本定义** - 这个概念本质上是什么？不用类比，直接定义
2. **核心要素** - 构成这个概念的必要组件有哪些？
3. **逻辑链条** - 这些要素如何相互作用？展示因果关系
4. **心智模型** - 给出一个帮助理解的思维框架
5. **实际应用** - 在什么场景下使用？给出 2-3 个具体例子
6. **常见误解** - 人们容易搞错的地方是什么？
7. **要点总结** - 用 5 条 bullet point 概括核心内容

## 要求

- 术语首次出现时必须定义
- 每一层解释要能独立理解
- 避免跳跃式推理
- 优先使用精确的技术语言而非模糊的类比
