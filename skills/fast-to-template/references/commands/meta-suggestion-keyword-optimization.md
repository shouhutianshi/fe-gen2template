# 提示词优化器

重写提示词，最大化输出质量。

## 优化维度

| 维度   | 检查点                     |
| ------ | -------------------------- |
| 清晰度 | 是否有歧义？指令是否明确？ |
| 结构   | 是否有清晰的输出格式？     |
| 约束   | 边界条件是否定义？         |
| 深度   | 是否引导深度思考？         |
| 具体性 | 是否足够具体？             |
